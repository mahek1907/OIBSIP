# Oasis Infobyte Data Science Internship

This repository contains the projects completed as part of the Oasis Infobyte Internship Program.

## Tasks Completed

### Task 1: Iris Flower Classification

Built a machine learning model to classify Iris flowers into Setosa, Versicolor, and Virginica species based on their measurements.

### Task 2: Unemployment Analysis with Python

Performed data analysis and visualization to study unemployment trends and understand the impact of various factors on unemployment rates.

### Task 3: Sales Prediction using Python

Developed a machine learning model to predict product sales based on advertising expenditures across different media channels.
